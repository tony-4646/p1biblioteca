Bienvenidos al sistema de gestión de la Biblioteca San Bartolo versión 1.0! Un programa hecho en visual studio code basado en la arquitectura MVC donde se podrá insertar, editar y ver usuarios como libros existentes, al igual que te permitirá realizar y finalizar préstamos de libros (NOTA: Se toma la fecha actual cuando se presta y se devuelve).

Para montar el programa necesitas usar el archivo adjunto de la base de datos, importarlo en tu gestor y ASEGURARTE de que en la clase conexion.php estén con los parámetros correctos, ya que se usa XAMPP y no involucra contraseña, por si usas otros como MAMPP que si la usen, corrige la sección de la contraseña.

El programa permite verificar cuando un libro ya está prestado directamente desde la base de datos, en esta primera versión no se incluyen métodos de eliminación, pero el resto del CRUD se puede realizar con normalidad. Pruebalo ya!
